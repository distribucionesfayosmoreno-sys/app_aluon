# app-aluon (Vite + React)

## Deploy en Vercel

Configura el proyecto con:
- **Root Directory**: `web`
- **Build Command**: `npm run build`
- **Output Directory**: `dist`
- **Install Command**: `npm install`

Este proyecto usa React Router (SPA). Se incluye `web/vercel.json` con rewrites para que rutas como `/detail` y `/checkout` funcionen en producción.
